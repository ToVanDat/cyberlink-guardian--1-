# CyberLink Guardian

CyberLink Guardian is a web security application built with React and TypeScript that uses AI to analyze the safety of URLs. The system focuses on detecting phishing and malicious links using prompt engineering instead of traditional static databases.

---

## 🚀 Features

- **AI-Powered URL Analysis**: Integrates Google Gemini AI to evaluate potential risks of URLs  
- **Structured AI Response**: Uses JSON Schema to ensure consistent and reliable output from the AI model  
- **Real-time Security News**: Aggregates and summarizes cybersecurity news using AI  
- **Persistent Storage**: Stores scan history using LocalStorage  
- **Responsive UI/UX**: Built with Tailwind CSS, supports modern dark theme and user feedback states (loading, error)  
- **Web Share Integration**: Allows users to share security alerts easily  

---

## 🛠️ Tech Stack

- **Frontend:** React, TypeScript  
- **Build Tool:** Vite  
- **AI:** Google Gemini API (@google/genai)  
- **Styling:** Tailwind CSS  
- **State Management:** React Hooks
